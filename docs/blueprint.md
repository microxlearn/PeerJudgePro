# **App Name**: PeerJudge

## Core Features:

- Create Room: Create a unique judging room with a generated room code and admin role.
- Join Room: Allow users to join a judging room by entering a room code and their name, with validation to prevent errors.
- Real-Time Scoring Display: Visually show results of score calculations, reflecting duplicate vote prevention.
- Admin Dashboard Redirect: Automatically redirect admins to a dashboard after room creation to manage the session.
- Mobile-Friendly UI: Adaptive layout using flexbox/grid for optimal judging via mobile phones
- Data Sanitization: Implement security measures to validate room codes, sanitize user inputs and prevent empty form submissions

## Style Guidelines:

- Primary color: A moderate blue (#4B8BBE) to inspire trust and academic seriousness.
- Background color: Off-white (#F9F9F9) to provide a clean and professional look.
- Accent color: A vibrant periwinkle (#736CED) for interactive elements like buttons and links.
- Body and headline font: 'Inter' (sans-serif) for clear and accessible typography.
- Minimal, line-based icons to represent each feature in the trust indicators section.
- Center-focused layout with soft shadows and rounded components for a modern and approachable feel.
- Subtle hover animations on trust indicator cards to provide user feedback and engagement.
