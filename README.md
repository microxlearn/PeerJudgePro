# PeerJudge

**PeerJudge** is a modern web platform designed to enable fair and transparent peer evaluation. The system allows users to submit work, review submissions from peers, and receive structured feedback in a collaborative environment.

---

## Overview

PeerJudge simplifies the process of peer assessment by providing a streamlined interface for submissions, evaluations, and feedback management. It is designed for academic environments, coding communities, and collaborative learning platforms where peer review plays an important role.

PeerJudge ensures that evaluations remain structured, consistent, and fair while encouraging constructive feedback between participants.

---

## Features

Submission Management
Users can submit their work or assignments through the platform for evaluation.

Peer Review System
Participants review and evaluate submissions from other users based on defined criteria.

Scoring and Feedback
Reviewers can provide ratings, comments, and structured feedback.

Transparent Evaluation
The system helps maintain fairness and accountability in the review process.

User-Friendly Interface
A clean and intuitive interface designed for smooth user interaction.

## Installation

Clone the repository

```
git clone https://github.com/microxlearn/PeerJudgePro.git
```

Navigate to the project directory

```
cd peerjudge
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:3000
```
